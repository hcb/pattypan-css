About

pattypan css is an amalgamation of different CSS frameworks/libraries/boilerplate I've liked over the years, primarily Skeleton (pattypan uses many of the same class names as Skeleton), as well as Flexbox Grid and Bulma to a lesser extent. I use this as a starter for my personal projects.
